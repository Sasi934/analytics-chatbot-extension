# Analytics Chatbot Browser Extension

## Overview

This project is a **browser extension that integrates analytics insights with chatbot interaction**.
It allows users to quickly access analytics information and interact with a chatbot interface for productivity.

## Features

* Lightweight browser extension
* Interactive chatbot interface
* Analytics insights integration
* Dynamic UI components
* Quick access panel inside browser

## Tech Stack

* JavaScript
* HTML
* CSS
* Browser Extension APIs

## Installation

1. Clone the repository

```bash
git clone https://github.com/Sasi934/analytics-chatbot-extension
```

2. Open your browser and go to **Extensions**

3. Enable **Developer Mode**

4. Click **Load Unpacked**

5. Select the project folder

## How It Works

* The extension injects UI elements into the browser.
* Users can interact with a chatbot.
* The chatbot provides analytics-related insights and suggestions.
